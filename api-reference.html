    API Reference infer v0.3.0

    Modules

    Infer

    Infer is an inference engine that allows to declare logic based on data schemas (such as Ecto) in a central and concise way.

    Infer.DSL
    Infer.DSL.Add
    Infer.DSL.Bind
    Infer.DSL.BindAll
    Infer.DSL.Bound

    Represents a reference to a variable bound using Bind or BindAll. Instead of an Atom or String to reference the bound value as a whole, a List can be passed to get a nested value at the given path inside the bound variable.

    Infer.DSL.Count
    Infer.DSL.Iterator

    Represents an iterator inside a loop, e.g. MapWhile.

    Infer.DSL.MapWhile
    Infer.DSL.QueryAll
    Infer.DSL.Range
    Infer.DSL.Ref

    Represents a reference to another field. :path is always traversed from the root, i.e. the type where the rule is defined. An Atom or String instead of a List is treated as if it was a list with one element. En empty list [] references the subject itself, as a whole.

    Infer.Ecto.Query

    Functions to dynamically generate Ecto query parts.

    Infer.Ecto.Query.Builder

    Internal data structure to keep track of all context needed to translate complex Infer rules to Ecto queries.

    Infer.Ecto.Query.TranslationError
    Infer.Ecto.Schema

    See Infer.

    Encapsulates the main functionality of working with rules.

    Infer.Parser

    Receives the module attributes created by the macros in Infer.DSL and converts them into normalized aliases and lists of Infer.Rule structs.

    Infer.Parser.Token
    Infer.Preloader

    Determines preloads based on predicates to be inferred.

    Infer.Rule
